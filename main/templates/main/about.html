О приложении

Пиложение создано в рамках задачи "Итоговый проект (vo_PJ)" по дисциплине "II. Углубленное программирование на Python".

Назначение приложения:
Введение пользователем данных о расходах и доходах, хранение введённых данных, группировка данных по категориям и типам операций и вывод сгруппированных данных на страницу.

Реализован функционал:
- 5 страниц с общим шаблоном;
- 2 таблицы CSS стилей;
- 3 модели;
- форма введения данных;
- сохранение введённых в форму данных в БД;
- вывод введённых в форму данных на страницу приложения;
- вывод сгруппированных данных на страницу приложения, с имользованием Pandas.
